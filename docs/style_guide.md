# Hunter AI 内容工厂 - 风格指南

## 写作风格

### 核心原则：去除 AI 味

**"没有 AI 味" = 观点 + 真实经验 + 可操作**

### 1. 观点

- 文章必须有明确的立场
- 敢于说"我认为"、"这个方法不行"、"大多数人搞错了"
- 不要两边讨好、面面俱到

### 2. 真实经验

- 用"我踩过的坑"代替"最佳实践列表"
- 写具体的场景、具体的问题、具体的解决过程
- 读者要能感觉到"这个人真的干过这事"

### 3. 可操作

- 不是列抽象的任务清单
- 结合读者背景给出真正可上手的操作
- 问自己：读者看完这段，明天能开始做什么？

## 表达风格

- 口语化表达，像朋友聊天
- 敢用"坦白说"、"老实讲"、"大实话"这类词
- 可以自嘲、可以吐槽
- 结尾不喊口号，说点掏心窝子的话

## 手机阅读适配

- 每段控制在 3-5 行
- 重要观点单独成段，加粗突出
- 适当使用 emoji 增加可读性
- 使用空行增加段落间呼吸感

## 违禁词列表

以下词汇会降低文章质量，应避免使用：

### 标题党/夸张词（微信敏感）

这类词容易被微信判定为标题党，可能影响文章推荐：

- 震惊、必看、100%、绝对、史上最、惊呆了
- 不转不是中国人、独家曝光、内幕揭秘、重大发现
- 速看、疯传、刷爆朋友圈、火了、炸了、吓死人

### 虚假宣传词（法律风险）

这类词可能涉及虚假宣传，有法律风险：

- 权威专家、医学证实、科学证明
- 国家认证、官方认定
- 独家秘方、祖传配方、包治百病

### 营销诱导词（用户体验差）

这类词会让读者反感，降低文章质量感：

- 限时优惠、仅剩XX名额、不可错过
- 立即行动、错过后悔、最后机会、手慢无、赶紧抢

### AI 生成痕迹词（去 AI 化）

这类词是 AI 生成内容的典型特征，需要替换为更自然的表达：

- 首先（开头使用时显得机械）
- 其次（列举时显得刻板）
- 最后（结尾总结时显得套路）
- 总之、综上所述（生硬的总结词）
- 值得注意的是、需要指出的是（过度强调）
- 不难发现、显而易见（假设性陈述）
- 毋庸置疑、众所周知（过度肯定/预设立场）

## AI 常用词替换

系统会自动替换以下 AI 痕迹词：

| 避免使用 | 替换为 |
|---------|--------|
| 首先， | （删除） |
| 其次， | （删除） |
| 最后， | （删除） |
| 总之， | 说到底， |
| 综上所述， | 聊到这里， |
| 综上所述 | 说白了 |
| 值得注意的是 | 有意思的是 |
| 需要指出的是 | 说句实话 |
| 不难发现 | 你会发现 |
| 显而易见 | 明摆着的 |
| 毋庸置疑 | 没跑了 |
| 众所周知 | 大家都知道 |

## 人性化技巧

1. **增加反问句**："这不是创作，这是赌博"
2. **使用第一人称经验**："我也经历过"
3. **添加情绪词汇**："噩梦、挫败感、更狠的是"
4. **场景化开头**："盯着空白文档发呆"
5. **对比式表达**："❌ 泛泛的选题 vs ✅ 具体的选题"

## 排版规范

### 标题层级

- H1: 文章主标题（仅一个）
- H2: 主要章节标题
- H3: 子章节标题

### 视觉优化

- 关键信息加粗
- 适当留白，每段不超过 3 行
- 使用引用块（`>`）强调金句
- 列表化处理复杂信息

### 封面图规格

- **比例**: 2.35:1（超宽电影画幅）
- **像素**: 1920x817 或 2560x1089
- **风格**: 科技/商务/生活/艺术
- **内容**: 抽象概念图，避免文字
